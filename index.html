Conway's Game of Life w/ Adjustable Bounds and Cellular Age

Grid Size

Sets the dimenions of the grid to be Grid Size x Grid Size cells

Survival Bounds (Inclusive)

If a living cell has a number of neighbors within these bounds,
they will continue to live through to the next iteration

Birth Bounds (Inclusive)

If a dead cell has a number of neighbors within these bounds,
they will be alive (birthed) in the next iteration

Max Age

If a living cell has been alive for more than Max Age iterations,
they will be dead in the next iteration

Press Enter to Begin/Restart Simulation

Default settings are intialized to regular Conway's Game of Life
